# lwgpp - A LOOP, WHILE and maybe a GOTO interpreter

This project was implemented for the Aarhus University course SWAPK. 
The code might not align with perfect coding standards as the goal was to apply at least 75% of the course material in this project. 

```bash
$ git clone git@github.com:gwydd12/lwgpp.git
$ cd lwgpp/
$ cmake -S . -B build
$ cmake --build build/
$ cd build/
$ ./lwgpp
```
